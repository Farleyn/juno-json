# Juno JSON for C++

Juno is a lightweight, single-header C++ library for parsing and serializing JSON.  
It allows you to read, manipulate, and write JSON without external dependencies.

## Features

- Parse JSON strings into C++ objects (`Value`)
- Serialize C++ objects back to JSON strings
- Supports: null, bool, integer, float, string, array, object
- Simple and easy-to-use API
- C++17 compatible

## Installation

Just copy `juno.h` into your project and include it:

```cpp
#include "juno.h"
